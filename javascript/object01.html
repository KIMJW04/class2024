01. at()

"문자열".at(위치값)

  • 지정한 숫자(index)를 받아, 문자열에서 해당되는 인덱스의 요소를 반환합니다.
  • 대소문자를 구별합니다.
  • 문자열의 길이보다 큰 인덱스를 지정하면 undefined를 반환합니다
  • 음수인 경우 문자열의 오른쪽에서부터 인덱스를 셉니다.
{
    "javascript reference".at()       // j
    "javascript reference".at(0)      // j
    "javascript reference".at(1)      // a
    "javascript reference".at(2)      // v
    "javascript reference".at(3)      // a
    "javascript reference".at(4)      // s
    "javascript reference".at(5)      // c
    "javascript reference".at(-1)     // e
    "javascript reference".at(-2)     // c
    "javascript reference".at(-3)     // n
    "javascript reference".at(-4)     // e
    "javascript reference".at(-5)     // r
    "javascript reference".at(100)    // undefined
}

02. charAt()

"문자열".charAt(위치값)

  • 지정한 숫자(index)를 받아, 문자열에서 해당되는 인덱스의 단일문자를 반환합니다.
  • 대소문자를 구별합니다.
  • 만약 인덱스가 문자열 길이보다 크거나 음수인 경우 빈 문자열("")을 반환합니다.
{
    "javascript reference".charAt(0)   // j
    "javascript reference".charAt(5)   // s
    "javascript reference".charAt(-1)  // ""
    "javascript reference".charAt(100) // ""
}

03. charCodeAt()

"문자열".charCodeAt(위치값)

  • 지정한 숫자(index)를 받아, 문자열에서 해당되는 인덱스의 유니코드 정수 값을 반환합니다.
  • 만약 인덱스가 문자열 길이보다 크거나 음수인 경우 NaN을 반환합니다.
{
    "javascript reference".charCodeAt(0)   // 106
    "javascript reference".charCodeAt(5)   // 99
    "javascript reference".charCodeAt(-1)  // NaN
    "javascript reference".charCodeAt(100) // NaN
}

04. codePointAt()

"문자열".codePointAt(위치값)

  • 지정한 숫자(index)를 받아, 문자열에서 해당되는 인덱스의 유니코드 포인트 정수 값을 반환합니다.
  • 만약 인덱스가 문자열 길이보다 크거나 음수인 경우 undefined를 반환합니다.
{
    "javascript reference".codePointAt(0)   // 106
    "javascript reference".codePointAt(5)   // 99
    "javascript reference".codePointAt(-1)  // undefined
    "javascript reference".codePointAt(100) // undefined
}

05. concat()

"문자열".concat(추가문자열1, 추가문자열2, ...)

  • 매개변수로 전달된 문자열을 받아, 기존 문자열에 이어붙인 새로운 문자열을 반환합니다.
{
    "Hello".concat(" ", "world", "!")   // "Hello world!"
}

06. includes()

"문자열".includes(검색문자열, 시작인덱스)

  • 문자열 포함 여부를 검색하여, 포함되면 true를 반환하고, 그렇지 않으면 false를 반환합니다.
  • 시작인덱스는 선택적 매개변수로, 검색을 시작할 위치를 지정합니다. 기본값은 0입니다.
{
    "Hello world".includes("world")      // true
    "Hello world".includes("World")      // false
    "Hello world".includes("world", 6)   // true
}

07. indexOf()

"문자열".indexOf(검색문자열, 시작인덱스)

  • 문자열을 검색하여, 주어진 값과 일치하는 첫 번째 위치값(index)을 반환합니다. 일치하는 값이 없으면 -1을 반환합니다.
  • 시작인덱스는 선택적 매개변수로, 검색을 시작할 위치를 지정합니다. 기본값은 0입니다.
{
    "Hello world".indexOf("o")         // 4
    "Hello world".indexOf("o", 5)      // 7
    "Hello world".indexOf("z")         // -1
}

08. lastIndexOf()

"문자열".lastIndexOf(검색문자열, 시작인덱스)

  • 문자열을 역순으로 검색하여, 주어진 값과 일치하는 첫 번째 위치값(index)을 반환합니다. 일치하는 값이 없으면 -1을 반환합니다.
  • 시작인덱스는 선택적 매개변수로, 검색을 시작할 위치를 지정합니다. 기본값은 문자열의 마지막 위치입니다.
{
    "Hello world".lastIndexOf("o")         // 7
    "Hello world".lastIndexOf("o", 5)      // 4
    "Hello world".lastIndexOf("z")         // -1
}

09. localeCompare()

"문자열".localeCompare(비교대상문자열)

  • 문자열과 문자열을 비교하여, 수치(음수, 양수, 0)을 반환합니다.
{
    "apple".localeCompare("banana")    // -1
    "banana".localeCompare("apple")    // 1
    "apple".localeCompare("apple")     // 0
}

10. match()

"문자열".match(정규식)

  • 정규식을 사용하여 문자열을 검색하고, 일치하는 문자열을 배열로 반환합니다.
{
    "Hello 123".match(/[0-9]+/)     // ["123"]
}

11. matchAll()

"문자열".matchAll(정규식)

  • 정규식을 사용하여 문자열을 검색하고, 일치하는 모든 문자열을 배열로 반환합니다.
{
    [..."Hello 123".matchAll(/[0-9]+/)]     // [["123", index: 6, input: "Hello 123", groups: undefined]]
}

12. normalize()

"문자열".normalize()

  • 주어진 문자열을 정규화된 형태로 반환합니다.
{
    "\u1E9B\u0323".normalize()   // "ṩ̣"
}

13. padEnd()

"문자열".padEnd(길이, 채울문자)

  • 주어진 길이에 맞게 뒤 문자열을 채우고, 새로운 문자열을 반환합니다.
{
    "Hello".padEnd(8, "*")   // "Hello***"
}

14. padStart()

"문자열".padStart(길이, 채울문자)

  • 주어진 길이에 맞게 앞 문자열을 채우고, 새로운 문자열을 반환합니다.
{
    "Hello".padStart(8, "*")   // "***Hello"
}

15. repeat()

"문자열".repeat(반복횟수)

  • 주어진 횟수만큼 반복 복사하여, 새로운 문자열을 반환합니다.
{
    "Hello".repeat(3)   // "HelloHelloHello"
}

16. replace()

"문자열".replace(정규식 또는 문자열, 교체할문자열)

  • 문자열에서 특정 문자열을 교체하여, 새로운 문자열을 반환합니다.
{
    "Hello world".replace("world", "there")   // "Hello there"
}

17. replaceAll()

"문자열".replaceAll(정규식 또는 문자열, 교체할문자열)

  • 문자열에서 특정 문자열을 모두 교체하여, 새로운 문자열을 반환합니다.
{
    "Hello world".replaceAll("o", "*")   // "Hell* w*rld"
}

18. search()

"문자열".search(정규식)

  • 문자열을 검색하고, 위치값(숫자)을 반환합니다.
{
    "Hello world".search("world")     // 6
}

19. slice()

"문자열".slice(시작인덱스, 종료인덱스)

  • 시작 위치에서 종료 위치 값을 추출하여, 새로운 문자열을 반환합니다.
{
    "Hello world".slice(6, 11)     // "world"
}

20. substring()

"문자열".substring(시작인덱스, 종료인덱스)

  • 시작 위치에서 종료 위치 값을 추출하여, 새로운 문자열을 반환합니다.
{
    "Hello world".substring(6, 11)     // "world"

21. substr()

"문자열".substr(시작인덱스, 길이)

  • 시작 위치에서 길이 값을 추출하여, 새로운 문자열을 반환합니다.
{
    "Hello world".substr(6, 5)     // "world"
}

22. split()

"문자열".split(구분자, 제한개수)

  • 문자열을 구분자로 구분하고, 여러 개의 문자열(배열)을 반환합니다.
  • 제한개수는 선택적 매개변수로, 반환되는 배열의 최대 길이를 제한합니다.
{
    "Hello world".split(" ")     // ["Hello", "world"]
}
