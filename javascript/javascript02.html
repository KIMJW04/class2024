10. 배열 : 데이터 불러오기 : filter()

배열의 요소를 필터링하여 새로운 배열을 생성하는 filter() 메서드를 사용합니다. 이 메서드는 주어진 함수의 조건을 만족하는 요소들만을 추출하여 반환합니다.

        console.log(element);
    });
    
    // 인덱스 출력
    arr.filter(function (element, index) {
        console.log(index);
    });
    
    // 전체 배열 출력
    arr.filter(function (element, index, array) {
        console.log(array);
    });
    
    // 조건에 맞는 요소 추출
    const num = arr.filter(function (element) {
        return element > 500;
    });
    console.log(num);

    const num1 = arr.filter((element) => {
        return element > 500;
    });
    console.log(num1);

    const num2 = arr.filter(element => {
        return element > 500;
    });
    console.log(num2);

    const num3 = arr.filter(e => e > 500);

11. 배열 : 데이터 불러오기 : 배열 구조분해할당

각 코드는 JavaScript의 배열 구조분해할당(destructuring assignment)에 관한 예제들입니다. 이를 통해 배열의 요소를 분해하여 여러 변수에 할당하거나, 기본값을 설정하거나, 일부 요소를 무시하거나, 나머지 요소를 배열로 할당하는 등의 다양한 기능을 보여줍니다.

기본 변수 할당:
const arr = [100, 200, "javascript"];에서 배열을 선언하고,
[a, b, c] = arr;를 통해 배열의 각 요소를 a, b, c에 할당합니다.

선언에서 분리한 할당:
배열을 선언과 동시에 구조분해하여 변수에 할당합니다.

선언에서 기본 값 설정:
배열 요소가 부족할 때 기본값을 설정하여 할당합니다.

배열 일부 요소 무시:
필요하지 않은 배열 요소는 콤마로 처리하여 무시할 수 있습니다.

나머지 값 할당:
...rest 문법을 사용하여 나머지 요소를 배열로 할당할 수 있습니다.

변수 값 교환:
두 변수의 값을 서로 바꾸는데 사용됩니다.

다차원 배열 펼침연산자:
중첩된 배열을 펼쳐서 다중 변수에 할당할 수 있습니다.

{
    // 기본 변수 할당
    const arr = [100, 200, "javascript"];
    const [a, b, c] = arr;

    console.log(a); 
    console.log(b); 
    console.log(c); 
}

{
    // 선언에서 분리한 할당
    const [a, b, c] = [100, 200, "javascript"];
    console.log(a); 
    console.log(b); 
    console.log(c); 
}

{
    // 선언에서 기본 값 설정
    const [a, b, c = "javascript"] = [100, 200];
    console.log(a); 
    console.log(b); 
    console.log(c); 
}

{
    // 배열 일부 요소 무시
    const [a, , b = "javascript"] = [100, 200];
    console.log(a); 
    console.log(b); 
}

{
    // 나머지 값 할당
    const [a, b, ...rest] = [100, 200, "javascript", "jquery"];

    console.log(a);     
    console.log(b);     
    console.log(...rest);  
}

{
    // 변수 값 교환
    let a = 100, b = 200;
    [a, b] = [b, a];
    console.log(a);
    console.log(b);
}

{
    // 다차원 배열 펼침연산자
    const [a, [b, c]] = [1, [2, "javascript"]];

    console.log(a);
    console.log(b);
    console.log(c);
}

12. 배열 : 데이터 불러오기 : 배열 펼침 연산자(Spread Operator)

각 코드는 JavaScript의 배열 펼침 연산자(Spread Operator)를 사용하는 예제들입니다.
이를 통해 배열을 복사하거나 합치는 등의 다양한 기능을 보여줍니다.


배열 복사:
주어진 배열을 복사하여 새로운 배열을 생성합니다.

배열 합치기:
두 개 이상의 배열을 합쳐서 새로운 배열을 생성합니다.

함수 인자로 사용하기:
배열을 함수의 인자로 전달할 때 펼침 연산자를 사용하여 각 요소를 함수의 인자로 전달합니다.

문자열 배열로 변환하기:
문자열을 각 문자로 분리하여 배열로 변환합니다. 

{
    // 배열 복사
    const arr = [100, 200, "javscript"];
    const copy = [...arr];
        
    console.log(arr);       
    console.log(...arr);    
    console.log(copy);      
        
}

{
    // 배열 합치기
    const arr1 = [100, 200];
    const arr2 = [300, 400];
    const merged1 = [...arr1, ...arr2];
    const merged2 = [...arr1, "javascript", ...arr2, "jquery"];

    console.log(arr1.concat(arr2));     
    console.log(merged1);               
    console.log(merged2);               
}

{
    // 함수 인자로 사용하기
    function numbers(a, b, c) {
        return a + b + c;
    }
    const nums = [100, 200, 300];
    console.log(numbers(...nums));
}

{
    // 문자열 배열로 변환하기
    const text1 = "javascript";
    const text2 = [...text1];
    console.log(text1);
    console.log(text2);
}

13. 객체 : 데이터 불러오기 : 기본

객체에서 속성을 가져오는 기본적인 방법은 점 표기법(.)과 대괄호 표기법([])을 사용하는 것입니다. 점 표기법은 객체의 속성 이름이 변수와 일치하는 경우에 사용할 수 있습니다. 대괄호 표기법은 속성 이름이 변수와 일치하지 않거나, 동적으로 속성에 접근해야 할 때 사용합니다. 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }
    console.log(obj.a);
    console.log(obj.b);
    console.log(obj.c);

    console.log(obj["a"]);
    console.log(obj["b"]);
    console.log(obj["c"]);
}

18. 객체 : 데이터 불러오기 : hasOwnProperty()

hasOwnProperty() 메서드는 객체가 특정 속성을 직접 소유하고 있는지 여부를 확인합니다. 또한, in 연산자를 사용하여 속성의 존재 여부도 확인할 수 있습니다. 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }

    console.log(obj.hasOwnProperty("a"));   //true
    console.log(obj.hasOwnProperty("b"));   //true
    console.log(obj.hasOwnProperty("c"));   //true
    console.log(obj.hasOwnProperty("d"));   //false

    // in 연산자
    console.log("a" in obj);    //true
    console.log("b" in obj);    //true
    console.log("c" in obj);    //true
    console.log("d" in obj);    //false

    // 내가 원하는 데이터가 obj안에 있는지
}

20. 객체 : 데이터 불러오기 : 객체 구조 분해 할당

각 코드는 JavaScript의 객체 구조 분해 할당(Object Destructuring Assignment)에 관한 예제들입니다. 이를 통해 객체의 속성을 분해하여 변수에
할당하거나, 기본값을 설정하거나, 속성 이름을 변경하거나, 중첩된 객체에서 속성을 추출하는 등의 다양한 기능을 보여줍니다.

기본 객체 분해 할당:
주어진 객체를 분해하여 각 속성을 변수에 할당합니다.

기본 값 할당:
객체의 속성 중 기본값이 없는 경우에 대비하여 기본값을 설정할 수 있습니다.

새로운 이름으로 변경:
분해한 객체의 속성 이름을 변경하여 다른 변수에 할당할 수 있습니다.

중첩된 객체 분해 할당:
중첩된 객체에서 속성을 추출하여 변수에 할당할 수 있습니다. 

{
    // 기본 객체 분해 할당
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    };

    const { a, b, c } = obj;

    console.log(a);
    console.log(b);
    console.log(c);
}

{
    // 기본 값 할당
    const obj = {
        a: 100,
        b: 200,
    }
    const { a, b, c = "javascript" } = obj;
    console.log(a);
    console.log(b);
    console.log(c);
}

{
    // 새로운 이름으로 변경
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }
    const { a: x, b: y, c: z } = obj;
    console.log(x);
    console.log(y);
    console.log(z);
}

{
    // 중첩된 객체 분해 할당
    const obj = {
        a: 100,
        b: { c: 200, d: "javascript" }
    }
    const { a, b: { c, d } } = obj;

    console.log(a);
    console.log(c);
    console.log(d);
}

javascript

// 기본 값 할당

// 새로운 이름으로 변경

21. 객체 : 데이터 불러오기 : 객체 펼침연산자

해당 코드는 객체 펼침 연산자(Spread Operator)를 사용하여 객체를 합치는 예제입니다.


객체 obj1과 obj2가 주어집니다.
merge1과 merge2 객체를 생성합니다.
merge1: obj1과 obj2의 속성을 모두 포함합니다.
merge2: obj1과 obj2의 속성을 모두 포함하면서 추가적으로 새로운 속성(x, y)을 가집니다.
obj1, merge1, merge2의 내용을 출력합니다.
결과적으로, merge1은 obj1과 obj2의 속성을 모두 포함한 새로운 객체를 나타내며, merge2는 추가적인 속성을 포함한 새로운 객체를 나타냅니다.

{
    const obj1 = { a: 100, b: 200 }
    const obj2 = { c: 100, d: 200 }
    const merge1 = { ...obj1, ...obj2 };
    const merge2 = { ...obj1, x: "javascript", ...obj2, y: "jquery" };
    console.log(obj1);
    console.log(merge1);
    console.log(merge2);
}
결과 확인하기
{ a: 100, b: 200 }
{ a: 100, b: 200, c: 100, d: 200 }
{ a: 100, b: 200, x: "javascript", c: 100, d: 200, y: "jquery" }

22. 서버 데이터 불러오기 : XMLHttpRequest

해당 코드는 XMLHttpRequest를 사용하여 서버에서 데이터를 비동기적으로 불러오는 예제입니다.

XMLHttpRequest 객체를 생성합니다.
open() 메서드를 사용하여 HTTP 요청을 설정합니다. 이 경우 GET 메서드를 사용하여 지정된 URL에서 데이터를 가져옵니다.
onreadystatechange 이벤트 핸들러를 정의합니다. 이 핸들러는 서버로부터의 응답이 변할 때마다 호출됩니다.
응답의 readyState가 4이고, HTTP 상태 코드가 200일 때, 즉 요청이 성공했을 때, 받아온 데이터를 처리합니다.
응답이 성공적이지 않은 경우(HTTP 상태 코드가 200이 아닌 경우), 오류 메시지를 출력합니다.
send() 메서드를 사용하여 요청을 서버로 보냅니다.

{
    {
        var xhr = new XMLHttpRequest();
        xhr.open("GET", "https://webstoryboy.github.io/webs2024/json/gineungsaJC2011_05.json", true);

        xhr.onreadystatechange = function () {
            if (xhr.readyState === 4 && xhr.status === 200) {
                var responseData = xhr.responseText;
                var jsonData = JSON.parse(responseData);

                console.log(jsonData);
            } else if (xhr.readyState === 4) {
                console.error("데이터 불러오기 오류: " + xhr.status);
            }
        };

        xhr.send();
    }
}

23. 서버 데이터 불러오기 : fetch API

해당 코드는 Fetch API를 사용하여 서버에서 데이터를 비동기적으로 불러오는 예제입니다.

fetch() 함수를 사용하여 지정된 URL에서 데이터를 가져옵니다. 이 함수는 Promise를 반환합니다.
첫 번째 .then() 메서드에서는 서버로부터의 응답을 처리합니다. response.json() 메서드를 사용하여 응답을 JSON 형식으로 변환합니다.
두 번째 .then() 메서드에서는 JSON으로 변환된 데이터를 처리합니다. 이 데이터를 콘솔에 출력합니다.
.catch() 메서드에서는 데이터 가져오기 과정에서 발생한 오류를 처리합니다. 

{
    fetch("https://webstoryboy.github.io/webs2024/json/gineungsaJC2011_05.json")
        .then(response => {
            return response.json();
        })
        .then(data => {
            console.log(data);
        })
        .catch(error => {
            console.error("데이터 불러오기 오류:", error);
        })
}
