10. 배열 : 데이터 불러오기 : filter()

배열의 요소를 필터링하여 새로운 배열을 생성하는 filter() 메서드를 사용합니다. 이 메서드는 주어진 함수의 조건을 만족하는 요소들만을 추출하여 반환합니다. 

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    console.log("10. 배열 : 데이터 불러오기 : filter()");
    
    // 요소 출력
    arr.filter(function (element) {
        console.log(element);
    });
    
    // 인덱스 출력
    arr.filter(function (element, index) {
        console.log(index);
    });
    
    // 전체 배열 출력
    arr.filter(function (element, index, array) {
        console.log(array);
    });
    
    // 조건에 맞는 요소 추출
    const num = arr.filter(function (element) {
        return element > 500;
    });
    console.log(num);

    const num1 = arr.filter((element) => {
        return element > 500;
    });
    console.log(num1);

    const num2 = arr.filter(element => {
        return element > 500;
    });
    console.log(num2);

    const num3 = arr.filter(e => e > 500);
    console.log(num3);
}

13. 객체 : 데이터 불러오기 : 기본

객체에서 속성을 가져오는 기본적인 방법은 점 표기법(.)과 대괄호 표기법([])을 사용하는 것입니다. 점 표기법은 객체의 속성 이름이 변수와 일치하는 경우에 사용할 수 있습니다. 대괄호 표기법은 속성 이름이 변수와 일치하지 않거나, 동적으로 속성에 접근해야 할 때 사용합니다. 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }
    console.log(obj.a);
    console.log(obj.b);
    console.log(obj.c);

    console.log(obj["a"]);
    console.log(obj["b"]);
    console.log(obj["c"]);
}
결과 확인하기
100
200
javascript
100
200
javascript

14. 객체 : 데이터 불러오기 : Object.keys()

Object.keys() 메서드는 객체의 속성 이름들을 배열로 반환합니다. 이를 통해 객체에 포함된 모든 속성 이름을 순회하거나, 속성의 개수를 확인할 수 있습니다. 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }

    console.log(Object.keys(obj));
}
결과 확인하기
[ "a", "b", "c" ]

Object.values() 메서드는 객체의 속성 값들을 배열로 반환합니다. 이를 통해 객체에 포함된 모든 속성 값에 접근할 수 있습니다.

16. 객체 : 데이터 불러오기 : Object.entries()

Object.entries() 메서드는 객체의 속성들을 [키, 값]의 형태로 이루어진 배열로 반환합니다. 이를 통해 객체의 모든 속성과 해당 값을 쉽게 순회하거나 다른 방식으로 처리할 수 있습니다. 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }

    console.log(Object.entries(obj));
}
결과 확인하기
[ ["a", 100], ["b", 200], ["c", "javascript"] ]

17. 객체 : 데이터 불러오기 : Object.assign()

객체의 속성을 병합하기 위해 Object.assign() 메서드를 사용합니다. 이 메서드는 첫 번째 인수로 전달된 대상 객체에 다른 객체들의 속성을 병합합니다. 

{
    const obj1 = {
        a: 100,
        b: 200
    }
    const obj2 = {
        c: 300,
        d: 400
    }
    const obj3 = Object.assign(obj1, obj2);
    console.log(obj3);
}
결과 확인하기
{ a: 100, b: 200, c: 300, d: 400 }

18. 객체 : 데이터 불러오기 : hasOwnProperty()

hasOwnProperty() 메서드는 객체가 특정 속성을 직접 소유하고 있는지 여부를 확인합니다. 또한, in 연산자를 사용하여 속성의 존재 여부도 확인할 수 있습니다. 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }

    console.log(obj.hasOwnProperty("a"));   //true
    console.log(obj.hasOwnProperty("b"));   //true
    console.log(obj.hasOwnProperty("c"));   //true
    console.log(obj.hasOwnProperty("d"));   //false

    // in 연산자
    console.log("a" in obj);    //true
    console.log("b" in obj);    //true
    console.log("c" in obj);    //true
    console.log("d" in obj);    //false

    // 내가 원하는 데이터가 obj안에 있는지
}
결과 확인하기

19. 객체 : 데이터 불러오기 : for in

for in 루프를 사용하여 객체의 속성을 반복적으로 순회할 수 있습니다. 이를 통해 객체의 각 속성과 해당 값에 접근할 수 있습니다. 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }

    console.log("19. 객체 : 데이터 불러오기 : for in")
    for (let el in obj) {
        console.log(el + ":" + obj[el]);
    }
}
결과 확인하기
a:100
b:200
c:javascript
