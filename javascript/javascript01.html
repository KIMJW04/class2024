01. 변수 : 데이터 저장

자바스크립트에서 변수는 값을 저장하고 참조하는데 사용합니다. 변수는 사용하면 데이터를 저장하고 나중에 해당 데이터를 재사용 할 수 있습니다. 

{
        let x = 100;
        let y = 200;
        let z = "javascript";
        
        console.log(x);
        console.log(y);
        console.log(z);
}
결과확인하기
100
200

02. 변수 : 데이터 저장 + 데이터 변경

자바스크립트에서 변수는 데이터를 저장하고 변경하는 데 사용됩니다. 변수를 선언하고 값을 할당하면 해당 변수는 그 값에 대한 참조를 보유하게 됩니다. 이후에 변수의 값을 변경하면 변수가 새로운 값으로 업데이트됩니다. 

{
        let x = 100;
        let y = 200;
        let z = "javascript";
    
        x = 300;
        y = 400;
        z = "jquery";
    
        console.log(x);
        console.log(y);
        console.log(z);
}
결과확인하기
300
400
jquery

10. 배열 : 데이터 저장(여러개) : 표현방법6 : 다차원 배열 / 2차

다차원 배열은 배열 안에 다른 배열이 포함된 배열을 말합니다. 2차원 배열의 경우 배열 내에 배열을 포함시키는 것으로, 이를 통해 여러 데이터를 저장할 수 있습니다. 

{
        const arr = [100, [200, 300, [400, 500, [600]]], ["javascript", "jquery"]];

        console.log(arr[1][1]); //300
        console.log(arr[1][2][2][0]); //600
        console.log(arr[2][0]); //javascript
        console.log(arr[2][1]); //jquery
}

12. 배열 : 데이터 저장(여러개) : 표현방법8 : 배열펼심연산자

배열 펼침 연산자는 배열의 요소를 풀어 개별 요소로 분리하는 기능을 제공합니다. 이를 통해 배열을 다른 배열에 병합하거나 함수 호출 시 인수로 사용할 수 있습니다. 

{
        const arr1 = [100, 200, "javascript"];
        const arr2 = [300, 400, "jquery"];
        const arr3 = [...arr1, 500];
    
        console.log(arr1); // 100,200,javascript
        console.log(...arr1); // 100200javascript
        console.log(...arr1, ...arr2); //100200javascript300400jquery
        console.log(...arr3); //100200javascript500
}

        console.log(obj1);
        console.log(merge1);
        console.log(merge2);
}
결과확인하기
{ a: 100, b: 200 }
{ a: 100, b: 200, c: 100, d: 200 }
{ a: 100, b: 200, x: "javascript", c: 100, d: 200, y: "jquery" }

24. 연산자 정리

연산자들은 여러 카테고리로 나눌 수 있습니다. 주요 연산자들은 다음과 같습니다:


산술 연산자
+: 덧셈
-: 뺄셈
*: 곱셈
/: 나눗셈
%: 나머지


할당 연산자
=: 변수에 값 할당
+=, -=, *=, /= 등: 산술 연산과 할당을 동시에 수행


비교 연산자
==: 값이 같은지 비교
===: 값과 타입이 같은지 비교
!=, !==: 값이 다른지 비교


논리 연산자
&&: 논리 AND
||: 논리 OR
!: 논리 NOT


증감 연산자
++: 1을 증가시킴
--: 1을 감소시킴


비트 연산자
&: AND
|: OR
^: XOR
~: 보수
<<: 왼쪽 시프트
>>: 오른쪽 시프트


조건(삼항) 연산자
condition ? expr1 : expr2: 조건이 참이면 expr1, 그렇지 않으면 expr2 반환


그 외
typeof: 피연산자의 데이터 유형을 반환
instanceof: 객체의 프로토타입 체인에서 특정 생성자의 존재 여부를 확인
자바스크립트는 다양한 연산자를 지원하며, 이러한 연산자들을 조합하여 복잡한 동작을 수행할 수 있습니다.

25. 자료형 정리

자바스크립트에서의 기본적인 자료형에 대해 설명해드리겠습니다.
자바스크립트는 동적으로 타입을 결정하는 언어로, 변수에 할당되는 값에 따라 자료형이 결정됩니다.
다음은 자바스크립트에서 사용되는 주요 자료형들입니다.


숫자(Number):

정수나 소수를 나타냅니다.
예: 10, 3.14


문자열(String):

문자들의 시퀀스를 나타냅니다.
예: "Hello, world!", 'JavaScript'


불리언(Boolean):

참(True) 또는 거짓(False) 값을 나타냅니다.
예: true, false


undefined:

값이 할당되지 않은 변수를 나타냅니다.
예: let x;


null:

의도적으로 값이 없음을 나타냅니다.
예: let y = null;


객체(Object):

여러 속성(키-값 쌍)을 포함하는 컨테이너입니다.
예: { name: 'John', age: 30 }


배열(Array):

값들의 순서 있는 리스트입니다.
예: [1, 2, 3, 4, 5]


함수(Function):

일련의 문장들을 실행하는 코드 블록입니다.
예: function add(a, b) { return a + b; }



이 외에도 여러 자료형들이 있지만, 이들은 주요 자료형들에 속합니다. 자바스크립트는 동적 타이핑 언어이므로 변수의 자료형은 실행 시간에 결정되며,
동일한 변수가 다른 자료형의 값을 가질 수 있습니다. 이러한 특성 때문에 자바스크립트는 유연하고 다양한 응용 프로그램을 작성하는 데 사용됩니다.
