01. 변수 : 데이터 저장

자바스크립트에서 변수는 값을 저장하고 참조하는데 사용합니다. 변수는 사용하면 데이터를 저장하고 나중에 해당 데이터를 재사용 할 수 있습니다. 

{
        let x = 100;
        let y = 200;
        let z = "javascript";
        
        console.log(x);
        console.log(y);
        console.log(z);
}
결과확인하기
100
200
javascript

02. 변수 : 데이터 저장 + 데이터 변경

자바스크립트에서 변수는 데이터를 저장하고 변경하는 데 사용됩니다. 변수를 선언하고 값을 할당하면 해당 변수는 그 값에 대한 참조를 보유하게 됩니다. 이후에 변수의 값을 변경하면 변수가 새로운 값으로 업데이트됩니다. 

{
        let x = 100;
        let y = 200;
        let z = "javascript";
    
        x = 300;
        y = 400;
        z = "jquery";
    
        console.log(x);
        console.log(y);
        console.log(z);
}
결과확인하기
300
400
jquery

03. 변수 : 데이터 저장 + 데이터 변경 + 데이터 추가

자바스크립트에서 변수는 데이터를 저장하고 변경하는 데 사용됩니다. 변수를 선언하고 초기값을 할당하면 해당 변수는 그 값을 저장하게 됩니다. 이후에 변수의 값을 변경하거나 추가할 수 있습니다. 

{
        let x = 100;
        let y = 200;
        let z = "javascript";
    
        x += 300;
        y += 400;
        z += "jquery";
    
        console.log(x);
        console.log(y);
        console.log(z);
}
결과확인하기
400
600
javascriptjquery

04. 상수 : 데이터 저장 + 데이터 변경(X)

자바스크립트에서 상수(constant)는 한 번 할당되면 그 값을 변경할 수 없는 변수입니다. 즉, 한 번 값을 할당하면 그 값을 변경하거나 재할당할 수 없습니다. 상수를 사용하면 일부 데이터를 변경하지 않고 유지할 수 있습니다. 

{
        const x = 100;
        const y = 200;
        const z = "javascript";
    
        // x += 300;
        // y += 400;
        // z += "jquery";
    
        console.log(x);
        console.log(y);
        console.log(z);
}
결과확인하기
100

05. 배열 : 데이터 저장(여러개) : 표현방법1 : 배열 선언

자바스크립트에서 배열(Array)은 여러 개의 데이터를 하나의 변수에 저장할 수 있는 자료구조입니다. 배열을 사용하면 여러 개의 값을 순서대로 저장하고 관리할 수 있습니다. new Array() 생성자를 사용하여 배열을 선언하고 초기화합니다. 

{
        const arr = new Array();
    
        arr[0] = 100;
        arr[1] = 200;
        arr[2] = "javascript";
    
        console.log(arr[0]);
        console.log(arr[1]);
        console.log(arr[2]);
}

100
200

09. 배열 : 데이터 저장(여러개) : 표현방법5 : push() 매서드를 이용한 배열 방법

배열의 끝에 새로운 요소를 추가하려면 push() 메서드를 사용합니다. 이 방법을 이용하여 배열에 데이터를 저장할 수 있습니다.

10. 배열 : 데이터 저장(여러개) : 표현방법6 : 다차원 배열 / 2차

다차원 배열은 배열 안에 다른 배열이 포함된 배열을 말합니다. 2차원 배열의 경우 배열 내에 배열을 포함시키는 것으로, 이를 통해 여러 데이터를 저장할 수 있습니다. 

{
        const arr = [100, [200, 300, [400, 500, [600]]], ["javascript", "jquery"]];
    
        console.log(arr[0]);    //100
        console.log(arr[1][0]); //200
        console.log(arr[1][1]); //300
        console.log(arr[1][2][2][0]); //600
        console.log(arr[2][0]); //javascript
        console.log(arr[2][1]); //jquery
}
결과확인하기
100
200
300
600

11. 배열 : 데이터 저장(여러개) : 표현방법7 : 배열구조분해할당

배열 구조 분해 할당은 배열의 각 요소를 배열에서 추출하여 변수에 할당하는 기능입니다. 이를 통해 배열에 저장된 데이터를 개별 변수에 쉽게 할당할 수 있습니다. 

{
        const arr = [100, 200, "javascript"];
    
        const [a, b, c] = arr;
    
        console.log(a);    //100
        console.log(b);    //200
        console.log(c);    //javascript
}
결과확인하기
100
200

12. 배열 : 데이터 저장(여러개) : 표현방법8 : 배열펼심연산자

배열 펼침 연산자는 배열의 요소를 풀어 개별 요소로 분리하는 기능을 제공합니다. 이를 통해 배열을 다른 배열에 병합하거나 함수 호출 시 인수로 사용할 수 있습니다. 

{
        const arr1 = [100, 200, "javascript"];
        const arr2 = [300, 400, "jquery"];
        const arr3 = [...arr1, 500];
    
        console.log(arr1); // 100,200,javascript
        console.log(...arr1); // 100200javascript
        console.log(...arr1, ...arr2); //100200javascript300400jquery
        console.log(...arr3); //100200javascript500
}
결과확인하기
[100, 200, "javascript"]
100200javascript
100200javascript300400jquery
100200javascript500
